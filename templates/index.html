About

Track Real-Time Stock Market Data

Welcome to the Stock Market Tracker! This application provides real-time data on stock market movers, allowing you to stay updated on the latest market trends. The table below shows top gainers (green) and losers (red), sorted by percentage change.

Why Sign Up?
  • Track your favorite stocks by adding custom symbols
  • Create a personalized watchlist for monitoring
  • Access your account from any device
Features
  • Real-time data from Yahoo Finance
  • Automatic 67-second refresh
  • Color-coded gains and losses
  • Responsive design for all devices

Learn more about this project and its tech stack on the About page.
