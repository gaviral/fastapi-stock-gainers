About the Stock Market Tracker

A modern dashboard for tracking stock market gainers and losers, built with FastAPI

Project Overview

This project is a web-based stock market tracker that displays real-time data on the top-performing stocks (gainers) and the worst-performing stocks (losers) in the market. It's designed to provide users with a quick overview of market movements and allow registered users to track their favorite stocks.

The data is automatically refreshed every 2 minutes to ensure you're getting up-to-date information. The application leverages the Yahoo Finance API to fetch real-time market data.

User Features

  • Password reset functionality via email
  • Responsive design for mobile and desktop

How It Works

Data Collection

The application uses the Yahoo Finance API to fetch real-time data about stock prices, changes, and other market information. This data is collected for all the major market indices and is updated every 2 minutes.

Data Processing

Once data is collected, it's processed to identify the top gainers and losers based on percentage change. The application also calculates additional metrics and formats the data for display in the dashboard.

User Customization

Registered users can create a personalized watchlist by adding stock symbols they want to track. This information is stored in the database and associated with their user account, allowing them to access their watchlist from any device.

Display & Visualization

The processed data is displayed in an intuitive dashboard that shows the top gainers and losers in the market. Users can easily identify trends with color-coded indicators showing positive (green) and negative (red) changes.

Getting Started

  1. Create an account - Sign up to access personalized features like watchlists.
  2. Add your favorite stocks - After logging in, add stock symbols to your watchlist to track them.
  3. Monitor market trends - Use the dashboard to see the top gainers and losers in real-time.
  4. Check back regularly - Data is updated every 2 minutes to give you fresh insights.
