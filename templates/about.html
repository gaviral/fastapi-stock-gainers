Project Overview

The Stock Market Tracker is a full-stack web application designed to display real-time stock market data, including top gainers and losers. This project demonstrates modern web development practices with a focus on real-time data processing, user authentication, and responsive design.

The application allows users to:

  • View real-time stock market data with automatic updates every 120 seconds
  • Create an account and log in securely
  • Recover forgotten passwords via email
  • Toggle password visibility for easier input
  • Track custom stock symbols that interest them
  • Monitor market changes with color-coded visual indicators

Tech Stack

Frontend

  • HTML5/CSS3
  • Jinja2 Templates

  • Font Awesome

Authentication

  • FastAPI Users
  • JWT Tokens
  • Secure Password Hashing
  • Session Management
  • Password Recovery

Data

  • Yahoo Finance API
  • yfinance Library
  • Async Data Processing
  • Real-time Updates
